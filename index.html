3 Pilar Pembentuk Sebuah Website

Sebuah website tidak semerta merta langsung terbentuk menjadi suatu halaman yang bisa dilihat dan di baca. Dibalik layarnya sebuah website itu dibuat dengan 3 jenis bahasa utama yaitu HTML, CSS, dan Javascript. Ketiga bahasa ini memiliki peranan yang berbeda dalam pembentukan sebuah website berikut ini adalah penjelas detail lebih lanjutnya

HTML

HTML merupakan singkatan dari Hypertext Markup Language dia bukanlah sebuah bahasa pemrograman dia adalah bahasa untuk menandai sebuah konten-konten website dengan menggunakan markup yang ditulis dengan symbol "<> </>" yang disebut dengan Tag html memiliki banyak sekali tag mulai dari tag tag untuk pemformatan text, tag pengelompokan, tag metadata, dan lain lain

dengan ditandainya sebuah text di dalam tag maka browser akan merender teks tersebut sebagaimana dia ditulis dibungkus menggunakan apa contohnya jika sebuah text dibungkus menggunakan tag header {h1 ~ h6} maka text tersebut akan ditampilkan sebagai judul pada konten.

CSS

Cascade Style Sheet atau CSS merupakan sebuah bahasa untuk mengatur bagaimana sebuah konten HTML akan ditampilkan. CSS memiliki banyak sekali penggaya yang bisa diterapkan pada sebuah element yang didefinisikan dengan bentuk property

Saat ini CSS sudah memiliki versi CSS3 yang dimana memiliki banyak sekali properti properti yang bisa memperindah tampilan website kita lengkapnya bisa di cek di MDN: CSS
